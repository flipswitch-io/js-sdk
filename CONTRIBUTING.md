# Contributing to Flipswitch JavaScript SDK

Thank you for your interest in contributing to the Flipswitch JavaScript SDK!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/js-sdk.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run tests: `npm test`
7. Commit your changes: `git commit -m "Add your feature"`
8. Push to the branch: `git push origin feature/your-feature`
9. Create a Pull Request

## Development Setup

Requirements:
- Node.js 18+
- npm 9+

```bash
# Install dependencies
npm install

# Run tests
npm test

# Type check
npm run typecheck

# Build
npm run build

# Run demo
npm run demo -- <your-api-key>
```

## Code Style

- Follow TypeScript best practices
- Use ESLint for linting: `npm run lint`
- Write tests for new functionality

## Pull Request Guidelines

- Keep changes focused and atomic
- Write clear commit messages
- Include tests for new functionality
- Update documentation as needed
- Ensure all tests pass and types check

## Reporting Issues

Please use GitHub Issues to report bugs or request features. Include:
- Node.js version
- Operating system
- Steps to reproduce
- Expected vs actual behavior
